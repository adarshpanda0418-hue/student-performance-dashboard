Student Performance Dashboard built using Python, Pandas, and Matplotlib to analyze student data and visualize insights.
📊 Student Performance Dashboard

A simple yet powerful data analysis project built using Python, Pandas, and Matplotlib to analyze student performance and visualize insights.

🚀 Features
📈 Calculate average marks
🏆 Identify top-performing student
📊 Filter students above average
🧮 Assign grades automatically
📉 Visualize grade distribution (bar chart)
🛠️ Tech Stack
Python 🐍
Pandas (Data Analysis)
Matplotlib (Visualization)
📂 Project Structure
student-performance-dashboard/
│
├── student_dashboard.py   # Main project file
├── project.py             # Additional logic/testing
├── README.md              # Project documentation
└── .gitignore            # Ignored files
▶️ How to Run
Clone the repository:
git clone https://github.com/adarshpanda0418-hue/student-performance-dashboard.git
Navigate to the folder:
cd student-performance-dashboard
Install required libraries:
pip install pandas matplotlib
Run the project:
python student_dashboard.py
📊 Sample Output
Average Marks displayed
Topper name and marks
List of students above average
Full dataset printed
Bar chart showing grade distribution
